# Security Policy

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, report them via GitHub's private vulnerability reporting:
[https://github.com/FlowScale-AI/flowscale-aios/security/advisories/new](https://github.com/FlowScale-AI/flowscale-aios/security/advisories/new)

Or email: **hello@flowscale.ai** with the subject line `[SECURITY] <brief description>`.

Please include:
- A description of the vulnerability and its potential impact
- Steps to reproduce
- Any suggested fix (optional)

We will respond within **72 hours** and aim to release a fix within **14 days** for confirmed issues.

## Scope

This policy applies to the FlowScale AIOS codebase at [github.com/FlowScale-AI/flowscale-aios](https://github.com/FlowScale-AI/flowscale-aios).

Note: FlowScale AIOS communicates with a locally-running ComfyUI instance. It is designed for local/trusted-network use. Do not expose the web app or ComfyUI publicly without additional authentication.
